# OmniSign Desktop

OmniSign Desktop is a graphical application for signing, validating, and re-timestamping PDF
documents. Built with Compose Multiplatform, it runs natively on Linux, Windows, and macOS.

### Features

- **Sign** PDF documents with PAdES BASELINE B / B-LT / B-LTA
- **Validate** electronically signed PDFs (including eIDAS EU LOTL validation)
- **Re-timestamp** existing signatures for digital continuity (archiving)
- **Extend** signatures to higher PAdES levels (B-T → B-LT → B-LTA)
- **Certificate discovery** from PKCS#12 files and hardware tokens (PKCS#11)
- **Export** validation reports in TXT, JSON, and multiple XML formats
- **Build** custom ETSI Trusted Lists for non-EU certificate authorities
- **Configuration profiles** for different signing workflows
- **Automatic renewal scheduling** via OS-native schedulers (Task Scheduler / cron / launchd)
- Wide range of [hash and encryption algorithms](algorithms.md) — SHA-2, SHA-3, Whirlpool, RIPEMD-160, RSA, ECDSA, EdDSA, and more
