# OmniSign Desktop

OmniSign Desktop is a graphical application for signing, validating, and re-timestamping PDF
documents. Built with Compose Multiplatform, it runs natively on Linux, Windows, and macOS.

## Quick start

### Features

- **Sign** PDF documents with PAdES BASELINE B / B-T / B-LT / B-LTA
- **Validate** electronically signed PDFs (including eIDAS EU LOTL validation)
- **Re-timestamp** existing signatures for digital continuity (archiving)
- **Certificate discovery** from PKCS#12 files and hardware tokens (PKCS#11)
- Support for SHA-256, SHA-384, and SHA-512 hash algorithms
