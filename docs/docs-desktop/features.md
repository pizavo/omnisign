# Features

| Feature                             | Supported |
|-------------------------------------|:---------:|
| Open and view PDF documents         |     ✅     |
| Page navigation and zoom            |     ✅     |
| Sign PDF documents (PAdES B–LTA)    |     ✅     |
| Validate signed PDFs (PAdES B–LTA)  |     ✅     |
| Timestamp extension (B-T/LT/LTA)    |     ✅     |
| Export validation report            |     ✅     |
| Configuration profiles (CRUD)       |     ✅     |
| Settings dialog                     |     ✅     |
| Trusted certificates panel          |     ✅     |
| Custom trusted list builder         |     ✅     |
| Archival renewal scheduling         |     ✅     |
| PKCS#11 hardware token support      |     ✅     |
| PKCS#12 file-based certificates     |     ✅     |
| Dark / light theme toggle           |     ✅     |
| JBR custom title bar (native frame) |     ✅     |

## Keyboard shortcuts

| Action        | Shortcut           |
|---------------|--------------------|
| Open file     | `Ctrl+O` / `⌘+O`   |
| Previous page | `←` or `Page Up`   |
| Next page     | `→` or `Page Down` |
| Zoom in       | `Ctrl++` / `⌘++`   |
| Zoom out      | `Ctrl+-` / `⌘+-`   |
| Reset zoom    | `Ctrl+0` / `⌘+0`   |
| Toggle theme  | Toolbar button     |
