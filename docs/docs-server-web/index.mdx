# OmniSign Web & Server

OmniSign Server is a Ktor-based HTTP server paired with a Kotlin/Wasm web frontend. It provides
the same signing, validation, and re-timestamping capabilities as the desktop and CLI apps, but
accessible through a browser — ideal for organisation-wide deployment.

## Architecture

| Component | Technology         | Purpose                                       |
|-----------|--------------------|-----------------------------------------------|
| Server    | Ktor (Kotlin/JVM)  | REST API for signing, validation, archiving    |
| Web       | Compose (Kotlin/Wasm) | Browser UI consuming the server API         |
