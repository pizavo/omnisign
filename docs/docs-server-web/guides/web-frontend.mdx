# Web Frontend

The OmniSign web frontend is built with Kotlin/Wasm using Compose Multiplatform. It provides
a browser-based PDF viewer that communicates with the Ktor server backend for signing and
validation operations.

## Building

import OsTabs from '@site/src/components/OsTabs';

## Browser compatibility

The Wasm frontend requires a browser with **WebAssembly** and **WebAssembly GC** support:

| Browser          | Minimum version |
|------------------|-----------------|
| Chrome / Edge    | 119+            |
| Firefox          | 120+            |
| Safari           | 18.2+           |

## Current capabilities

| Feature                          | Available |
|----------------------------------|:---------:|
| Open and view PDF documents      |     ✅     |
