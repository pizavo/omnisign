# OmniSign CLI

OmniSign CLI is a command-line tool for signing, validating, and re-timestamping PDF documents.
It supports PAdES BASELINE B, B-T, B-LT, and B-LTA signature levels and runs on Linux, Windows,
and macOS.

### Signing a document

```bash
omnisign sign --level B-T --hash SHA256 --keystore my-key.p12 document.pdf
```

### Validating a document

```bash
omnisign validate document-signed.pdf
```
